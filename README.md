# Nomad Racers — F1 STEM Racing Team

Official website for Nomad Racers, an F1 STEM Racing competition team from Haileybury Almaty, Almaty, Kazakhstan.

## About
Nomad Racers is a student engineering team competing in the F1 STEM Racing Competition. We design, simulate, and manufacture a miniature F1 car from scratch — combining aerodynamic engineering, CAD modelling, CAM manufacturing, and enterprise strategy.
